How it works

仕組み

Chatterbox TTSは「参照音声の声質」と「テキスト（セリフ）」を組み合わせて新しい音声を生成します。

処理フロー
参照音声
wav/
→
Chatterbox TTS
AI合成エンジン
→
生成音声
output/
覚えておくべきポイント 参照音声が「どんな声か」を、テキストが「何を喋るか」を決めます。生成エンジンは確率的に動作するため、同じ設定でも毎回異なる結果が出ます。

Setup

セットアップ手順

1
Pythonのインストール
Python 3.11.9 をダウンロードしてインストールします。
初心者向け：PATH設定を忘れずに インストーラーの最初の画面で「Add python.exe to PATH」のチェックボックスを必ずオンにしてください。これをオフのままにすると以降のセットアップが全て失敗します。

別バージョン（3.12等）が既にインストールされている場合、パスの優先順位で意図しないバージョンが使われることがあります。コマンドプロンプトで python --version を実行して確認してください。
2
初期セットアップ
01_setup.bat をダブルクリックして実行します。仮想環境の作成とライブラリのインストールが行われます。所要時間は約5〜10分です。
初心者向け：WARNINGは気にしなくてOK インストール中に赤い文字の警告（WARNING）が出ることがありますが、ERROR でなければ基本的に問題ありません。
初心者向け：フォルダのパスに注意 プロジェクトフォルダを日本語や全角文字を含むパスに置かないでください（例：C:\Users\太郎\デスクトップ\）。英数字のみのパスでないとエラーが発生する場合があります。
3
参照音声ファイルの配置
wav/ フォルダに参照音声ファイルを配置します。次の「参照音声の準備」セクションを参照してください。
Reference Audio

参照音声ファイルの準備

参照音声はAIが「声の雰囲気」を学習するための素材です。この品質が最終結果のクオリティに直結するため、丁寧に準備してください。

品質を上げるためのポイント

ポイント説明
1セリフ = 1ファイル複数セリフが連結されたファイルは使わない。1つのセリフだけが入ったWAVを用意する
ノイズを避けるBGMやSEが混入した音声は品質低下の原因。クリーンな音声を選ぶ
感情・トーンを合わせる叫び声の参照で穏やかなセリフを生成すると不自然になりやすい。生成したいセリフの雰囲気に近い参照音声を使う
短い音声を使う1〜5秒程度の短いセリフが最も安定した結果を得やすい
英数字のファイル名日本語・全角文字のファイル名はパスの問題でエラーになる場合がある
Usage

使い方

セリフの設定

generate_voice.py の LINES セクションを編集して、生成したいセリフを設定します。

ファイル上部
REF_CAPTAIN などの定数で参照WAVファイルのパスを定義
LINES セクション
生成するテキスト・参照ファイル・パラメータを辞書形式で定義
LINES = {
    "yes_sir": {
        "text": "Yes sir!",          # 生成するセリフ
        "ref":  REF_YESSIR,           # 参照音声（声の基準）
        "exaggeration": 0.5,         # 感情の強度
        "cfg":          0.5,         # 参照音声への忠実度
    },
    # "disabled": { ... },        ← # を付けると生成されない
}
初心者向け：カンマに注意 Pythonの辞書はカンマの位置が重要です。各 } の後にカンマ , を忘れると文法エラーになります。既存の行をコピペして編集するのが安全です。

音声を生成する

02_generate.bat をダブルクリックして実行します。初回はモデルファイル（数GB）のダウンロードが発生します。

30秒〜数分
CPUのみの場合（1行あたり）
数秒
GPU搭載の場合（NVIDIA RTX推奨）

結果を保存する

生成されたWAVファイルは output/ に自動保存されます。同じ名前のファイルは上書きされるため、気に入った結果は success/ へ移動してください。

ポイント 同じ設定でも実行するたびに異なる結果が出ます。納得のいく結果が出るまで何度か実行して、複数の候補から最も良いものを選ぶのがおすすめです。
Parameters

音声パラメータの調整

おすすめの始め方 まずは exaggeration=0.5、cfg=0.5 で試して、結果を聞きながら 0.1 ずつ調整していくと効率的です。
exaggeration 0.0 〜 1.0
感情表現の強度。高いほど活気があり大げさな読み上げになります。掛け声や笑い声は高めに、うめき声は低めにすると自然です。
0.0〜0.4 落ち着いた表現
0.5〜0.6 標準的な表現
0.7〜1.0 大げさな表現
cfg 0.0 〜 1.0
参照音声への忠実度。高いほど元の声に近くなりますが、高すぎると不自然になる場合もあります。
0.0〜0.3 AIの自由度が高い
0.4〜0.6 バランス型（おすすめ）
0.7〜1.0 元の音声に酷似
DST MOD Integration

DST MOD への反映

FMOD Designer での変換が必要です

生成したWAVファイルをそのままDSTのMODに組み込むことはできません。FMOD Designer というツールを使って、WAVファイルをDSTが読み込める形式（.fev + .fsb）に変換する必要があります。

FMOD Designerの使い方については、専用のFMOD Designerプロジェクトフォルダ内のREADMEを参照してください。

FAQ

よくある質問

別のキャラクターのMODを作りたい
このプロジェクトをフォルダごとコピーして、以下を差し替えてください。
1
wav/ の参照音声ファイルを対象キャラのものに入れ替える
2
generate_voice.py 上部の定数を新しいファイル名に合わせる
3
LINES のテキスト・参照・パラメータを書き換える
ファイル名を変えたい場合 スクリプト名（generate_voice.py）を変更した場合は、02_generate.bat 内のファイル名も合わせて変更してください。
英語以外のテキストは生成できる？
Chatterbox TTSは主に英語に最適化されています。他の言語では品質が低下する可能性がありますが、試す価値はあります。
GPUを使いたい
2つの変更が必要です。
1
01_setup.bat のPyTorchインストール行をGPU対応版に変更する（PyTorch公式サイトで確認）
2
generate_voice.py の device="cpu" を device="cuda" に変更する
Troubleshooting

トラブルシューティング

Pythonが見つからないというエラーが出た
Python 3.11がインストールされていないか、PATHに追加されていません。Python 3.11.9 をインストールし、「Add python.exe to PATH」にチェックを入れてください。
1
venv/ フォルダを削除する
2
01_setup.bat を再実行する
torchやライブラリ関連のエラーが発生した
仮想環境が破損している可能性があります。
1
venv/ フォルダを完全に削除する
2
01_setup.bat を再実行してセットアップをやり直す
「SyntaxError」が表示された
generate_voice.py の編集でPythonの文法エラーが発生しています。エラーメッセージの行番号付近を確認してください。
!
カンマ , の付け忘れ
!
ダブルクォーテーション " の閉じ忘れ
!
インデント（空白の数）がずれている
「Reference audio not found」というエラーが出た
wav/ フォルダに指定されたファイルが見つかりません。
1
ファイル名のスペルが generate_voice.py 上部の定数定義と完全に一致しているか確認
2
ファイルが wav/ フォルダ直下に置かれているか確認（サブフォルダに入っていないか）
生成処理が遅い
CPUのみでの生成は正常な動作で、1行あたり30秒〜数分かかります。高速化したい場合は NVIDIA GeForce RTX などのGPU導入を検討してください。
モデルのダウンロードが失敗した
ダウンロード用キャッシュが破損している可能性があります。
1
C:\Users\[ユーザー名]\.cache フォルダを削除する
2
02_generate.bat を再実行する
